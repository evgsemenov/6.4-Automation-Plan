## Тест-кейсы для проведения функционального тестирования отправки формы записи на курс ["Тестировщик ПО"](https://netology.ru/programs/qa) сайта [Netology.ru](https://netology.ru/)

### 1.1 Переход к странице профессии с главной страницы Нетологии через кнопку "Каталог курсов"
- Открыть сайт Netology.ru
- Нажать кнопку "Каталог курсов"
- В выпадающем меню навести курсор на раздел "Программирование"
- В выпадающем подменю кликнуть по кнопке "Тестировщик ПО"

  **Ожидаемый результат:**   
  Открывается страница профессии "Тестировщик ПО"

### 1.7 Запись на курс неавторизованного пользователя
- Открыть страницу профессии [Тестировщик](https://netology.ru/programs/qa), не авторизуясь на сайте
- Проскроллить страницу вниз до формы записи на курс
- В поле "Имя" ввести имя из тестовых данных
- В поле "Телефон" ввести телефон из тестовых данных
- В поле "электронная почта" ввести email из тестовых данных
- Нажать кнопку "Записаться"

  **Ожидаемый результат:**  
  Появляется сообщение об успешной записи на курс

### 1.11 Отправка пустой формы записи
- Открыть страницу профессии [Тестировщик](https://netology.ru/programs/qa)
- Проскроллить страницу вниз до формы записи на курс

  **Ожидаемый результат:**   
  Поля "Имя", "Телефон", "Электронная почта" не заполнены

- Нажать кнопку "Записаться"

  **Ожидаемый результат:**  
  Поля "Имя", "Телефон", "Электронная почта" подсвечены комментарием "Обязательное поле". Форма не отправлена

### 1.12 Валидация данных поля "Имя" при заполнении формы записи на курс профессии
- Открыть страницу профессии [Тестировщик](https://netology.ru/programs/qa)
- Проскроллить страницу вниз до формы записи на курс

- В поле "Имя" ввести кириллические символы не короче двух символов, с использованием символов "ё", "-", "Space"

  **Ожидаемый результат:**  
  Значение принимается, никаких комментариев под полем не появляется

- Нажать кнопку "Записаться"

  **Ожидаемый результат:**  
  Поля "Имя" подсвечивается с комментарием "Обязательное поле".

- В поле "Имя" ввести любые спецсимволы (кроме дефиса)

  **Ожидаемый результат:**  
  Поля "Имя" подсвечивается с комментарием "Должно состоять из букв"

- В поле "Имя" ввести любые цифры

  **Ожидаемый результат:**  
  Поля "Имя" подсвечивается с комментарием "Должно состоять из букв"
